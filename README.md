# 🧠 FluxNote

FluxNote is an AI-powered study assistant that transforms PDF documents into summaries, flashcards, quizzes, and interactive notes.

## ✨ Features

- 📄 Upload PDF
- 📝 AI Summary
- 🎴 Flashcards
- ❓ Quiz Generator
- 💬 Chat with PDF (Coming Soon)

## 🛠 Tech Stack

- Python
- Streamlit
- PyMuPDF
- Gemini API
